# zip

### 打包成tar.gz格式压缩包
```
tar -zcvf test.tar.gz /test
```

### 解压tar.gz格式压缩包
```
tar zxvf test.tar.gz
```

### 打包成tar.bz2格式压缩包
```
tar -jcvf renwolesshel.tar.bz2 /renwolesshel
```

### 解压tar.bz2格式的压缩包
```
tar jxvf renwolesshel.tar.bz2
```

### 压缩成zip格式
```
zip -q -r renwolesshel.zip renwolesshel/
```

### 解压zip格式的压缩包
```
unzip renwolesshel.zip
```
