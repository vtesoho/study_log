## limit(分页查询)

```
mysql 里面（表示从0开始取五条数据 ）
limit
    0, 5 
也可以直接写，这样的写法就是表示(0,5)
limit
    5


pgsql里面（表示从0开始取五条数据）
limit
    0
offset
    5
```
